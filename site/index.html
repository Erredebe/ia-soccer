IA Soccer

Simulador manager canallesco de fútbol en Node.js

¿Qué es esto?

IA Soccer es un experimento técnico creado como punto de partida para un simulador de manager futbolero con mucho descaro. Todo el motor está implementado en JavaScript/TypeScript y se ejecuta desde la línea de comandos.

Si has llegado hasta aquí desde un despliegue web, tranquilo: no estás perdido. Simplemente este proyecto no tiene una interfaz gráfica todavía, pero dejamos a mano la documentación para que puedas probarlo en tu máquina.

Cómo probarlo localmente

  1. Clona el repositorio o descarga el código.
  2. Instala dependencias (si las hubiera) y ejecuta el CLI con:
npm install
npm run demo

El comando abrirá una aventura canallesca en tu terminal, donde podrás fichar sinvergüenzas, negociar cláusulas y jugar partidos cargados de picante futbolero.
